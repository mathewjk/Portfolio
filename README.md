# mathewjk.com

Personal portfolio site for Mathew Joseph Kayyalackakom.

Static site — no build step. Just HTML/CSS/JS.

## Files
- `index.html` — the site
- `404.html` — custom not-found page
- `CNAME` — custom domain for GitHub Pages (mathewjk.com)
- `server.js` — tiny local preview server (not used in production)

## Local preview
```
node server.js
```
Then open http://localhost:4321

## Hosting
Deployed via GitHub Pages from the `main` branch. Custom domain: mathewjk.com.
